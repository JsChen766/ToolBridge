# Security Policy

ToolBridge can execute local functions declared by npm packages.

Use only trusted tool packages in your project.

Important behavior:

- ToolBridge does not auto-expose all installed packages.
- Only tools selected in `toolbridge.config.json` are exposed.

## Reporting a Vulnerability

Please report security issues through GitHub Security Advisories or by contacting the maintainer.

Do not publicly disclose unpatched vulnerabilities.
